# NexusAI — AI Customer Support Agent

A futuristic landing page for an AI-powered customer support agent, built with **React** and **Tailwind CSS**.

[Link](https://github.com/apps/celesto-anti-slop) to app.

## Features

- 🚀 **Hero section** with animated rotating headlines and live statistics
- ⚡ **Features grid** showcasing 6 core AI capabilities
- 💬 **Live chat demo** with animated AI conversation replay
- ⭐ **Testimonials** from real customers
- 💰 **Pricing tiers** (Starter, Growth, Enterprise)
- 📣 **CTA banner** and footer with navigation

## Tech Stack

- [React](https://react.dev/) — UI components
- [Tailwind CSS v4](https://tailwindcss.com/) — Utility-first styling
- [Vite](https://vite.dev/) — Build tool

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.

## Build

```bash
npm run build
```
